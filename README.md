# Personal Website

A simple static website built with a custom Python build system that compiles HTML content into JavaScript modules.

## Building

1. Navigate to the `content/` directory:
   ```bash
   cd content
   ```

2. Run the build script:
   ```bash
   python3 compile.py
   ```

3. Open `index.html` in a browser to preview.

4. Commit and push to GitHub to deploy (GitHub Pages serves automatically).

## File Organization

### Manually Maintained Files

**Content (edit these to change website content):**
- `content/*.html` - Page content files (e.g., `home.html`, `games.html`, `research.html`, `talks.html`)
- `content/content_list.txt` - Tab configuration (one line per tab: `[id] [Tab Name]`)

**Code (edit these to change website structure/behavior):**
- `content/compile.py` - Build script that generates JavaScript from HTML
- `index.html` - Main page template

**Static Assets:**
- `css/`, `js/`, `fonts/`, `images/`, `pdf/` - Static files served as-is

### Automatically Generated Files

**Do not edit these files manually** - they are regenerated by `compile.py`:
- `content/tab_list.js` - Navigation menu generation
- `content/tab_content.js` - Content loader script
- `content/*.js` - Compiled page content (e.g., `home.js`, `games.js`, `research.js`, `talks.js`)

## Adding a New Page

1. Add a line to `content/content_list.txt`: `[id] [Tab Name]`
2. Create `content/[id].html` with your page content
3. Run `python3 compile.py` in the `content/` directory
4. The new tab will appear in the navigation automatically
